This page is an example of usage for the npm package mortgage-calculator

I've created this package to learn how to code JavaScript, the code has been created following the TDD technique

With the mortgage-calculator package you can obtain the maximum payment you should be commiting to according to a risk factor, you can obtain the mortgage and the amortization table.

To see the details of the project you can go to the github page

You can play with the sliders to update live the results of the mortgage calculations or update the value directly on the input.

For this example I have used the standard values of the formula with a risk of 36%
